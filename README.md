The software KukaMini is to control the robot arm 3D printed

I designed mine using the esp8266, but you can choose any other board to do this.

The file braco_kuka is the code I put in the esp8266 that receive commands by udp protocol over wifi to movement each part of the robot

Complete files and guide to print your mini robot arm kuka:
https://cults3d.com/pt/modelo-3d/gadget/kuka-based-robotic-arm-with-servos-fusion-360-files

Little box to put the controller:
